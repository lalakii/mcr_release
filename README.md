# mcr_release
BIOS microcode update tool.
[Download](https://github.com/lalakii/mcr_release/releases)

Linux:

```bash
chmod +x lalaki_mcr

./lalaki_mcr -u "your_bios_file" -src "your_old_microcode" -dest "new_microcode"
```
Windows Powershell:
```powershell
.\lalaki_mcr -u "your_bios_file" -src "your_old_microcode" -dest "new_microcode"
```
Windows CMD:
```batch
lalaki_mcr -u "your_bios_file" -src "your_old_microcode" -dest "new_microcode"
```
